1.3 Combinations: The Binomial Theorem

Proof: As in the proof of the binomial theorem, the coefficient of $x_1^{n_1}x_2^{n_2}x_3^{n_3} \cdots x_t^{n_t}$ is the number of ways we can select $x_1$ from the $n_1$ of the $n$ factors, $x_2$ from $n_2$ of the $n-n_1$ remaining factors, $x_3$ from $n_3$ of the $n-n_1-n_2$ now remaining factors, ... , and $x_t$ from $n_t$ of the last $n-n_1-n_2-n_3-\cdots-n_{t-1}=n_t$ remaining factors. This can be carried out – as in part (a) of Example 1.22 – in $$ {n \choose n_1}{n-n_1 \choose n_2}{n-n_1-n_2 \choose n_3} \cdots {n-n_1-n_2-n_3- \cdots -n_{t-1} \choose n_t} $$ $$ \begin{equation} \begin{split} \displaystyle &= \bigg[\frac{n!}{n_1!\color{grey}{\cancel{(n-n_1)!}}}\bigg]\bigg[\frac{\color{grey}{\cancel{(n-n_1)!}}}{n_2!\color{grey}{\cancel{(n-n_1-n_2)!}}}\bigg]\bigg[\frac{\color{grey}{\cancel{(n-n_1-n_2)!}}}{n_3!\color{grey}{\cancel{(n-n_1-n_2-n_3)!}}}\bigg] \\ \\ &\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \cdots\bigg[\frac{\color{grey}{\cancel{(n-n_1-n_2-n_3- \cdots -n_{t-1})!}}}{n_t!(n-n_1-n_2-n_3- \cdots -n_{t-1}-n_t)!}\bigg] \\ &= \frac{n!}{n_1!n_2!n_3! \cdots (n_t!)(0!)} \ \succ \ n_1+n_2+n_3+\cdots+n_{t}=n \\ &= \frac{n!}{n_1!n_2!n_3! \cdots n_t!} = {n \choose n_1,n_2,n_3, \dots, n_t} \ \color{#aaa}{\text{:: multinomial coefficient}} \end{split} \end{equation} $$ ways. Note that when $t=2$ this reduces to a binomial coefficient. $\square$
